iPhone Wi-Fi Install

VOXOS can be built, installed, and launched on the paired iPhone through Xcode/CoreDevice when the phone is reachable on the same network.

Default Device​

  • Device: Shyam's iPhone
  • Device ID: E4A02640-BD46-5CCF-9926-494D1316F052
  • Bundle ID: com.shyamzzp.voxos
  • Development team: 2JLQM38KF6

Command​

Run:

./scripts/install_wifi_iphone.sh

The script builds the native Xcode target for iphoneos, installs the .app with xcrun devicectl, and launches VOXOS.

The Xcode project is configured with development team 2JLQM38KF6, so local Xcode builds and the Wi-Fi install script use the same signing team by default.

Faster Development Loop​

For repeated iPhone development installs, use the fast Debug helper:

./scripts/dev_iphone_fast.sh

This keeps a persistent DerivedData folder at ~/Library/Developer/Xcode/DerivedData/VOXOSFastDev, disables the CLI index store, and relies on Xcode incremental compilation instead of deleting the build directory every run.

To rebuild, install, and launch whenever Swift/project files change:

./scripts/dev_iphone_fast.sh --watch

The watch mode uses fswatch when installed and falls back to a one-second polling watcher. It is for development only; release/archive builds should keep using normal Xcode build/archive flows.

Overrides​

The script supports environment overrides:

DEVICE_ID=<device-id> DEVELOPMENT_TEAM=<team-id> ./scripts/install_wifi_iphone.sh

Use xcrun devicectl list devices to confirm that the iPhone is listed as available (paired) before installing.

Trust / Verification Error​

If the app installs but iOS says it cannot verify the app or account, the phone is blocking the development certificate before VOXOS starts. This is not an in-app internet failure.

On the iPhone:

  1. Open Settings.
  2. Go to General > VPN & Device Management.
  3. Open the Apple Development profile for this account/team.
  4. Tap Trust, then confirm.
  5. Confirm Developer Mode is enabled under Privacy & Security > Developer Mode.
  6. Keep the phone online once after trusting so iOS can verify the profile.

Then rerun:

./scripts/install_wifi_iphone.sh
