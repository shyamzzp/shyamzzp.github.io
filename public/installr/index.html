About the project

Installr

Installr is a governed installer-script marketplace. The product makes setup commands easier to discover, safer to copy, and easier to keep current through metadata validation and release guardrails.

Problem

Setup commands usually live in scattered READMEs, old notes, and copied shell snippets. Teams need command provenance, prerequisites, verification steps, rollback guidance, and stale-status visibility before running anything locally.

What the demo shows

  • Search and filters for category, platform, and trust state.
  • Installer cards with command, provenance, maintainer, and freshness.
  • Copy affordances plus install playbooks for verify and rollback steps.

Production shape

The live project uses React, TypeScript, Vite, catalog metadata validation, preflight command guardrails, URL-state tests, catalog verification artifacts, and release-readiness checks.
