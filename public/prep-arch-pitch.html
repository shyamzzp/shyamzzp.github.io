The Problem

Knowledge tools are
broken by design.

Professionals juggle dozens of tabs, docs, notes, and files daily. Existing tools force a rigid linear structure — or none at all.

🧩

Fragmented workflows

Teams constantly switch between docs, chat, storage, and task tools. Context is lost at every handoff. The average knowledge worker loses 2.5 hrs/day to tool-switching alone.

📄

Rigid document structure

Traditional docs are flat and sequential. Real thinking is hierarchical and associative. No mainstream tool maps to how humans naturally organise ideas.

🔌

No unified workspace

Files live in Drive, notes in Notion, tasks in Jira. There is no single surface where structured organisation, rich notes, and file attachments naturally coexist.

🤖

AI without workspace context

Existing AI tools operate on generic prompts. There is no product where the AI understands your personal workspace hierarchy, history, and relational structure.

The Solution

One workspace.
Infinite clarity.

Prep-Arch brings a spatial model where every piece of knowledge has a place — with AI, Google Drive, and real-time sync built in from day one.

🗂️

Workspaces & Topics CORE

Organise everything into collapsible workspaces with sub-topics, pinnable sections, and password-protected vaults. Full drag-and-drop reordering.

🪟

Floating Note Panels

Each item opens a draggable, resizable note panel. Run multiple panels simultaneously — like a real desktop OS, inside the browser.

✍️

Rich Notes Editor

Markdown-powered with autosave, formatting shortcuts, code blocks, and instant edit mode. Everything syncs to Supabase in real time.

📎

File Vault + OCR NEW

Drag-drop or paste images, ZIP batches, and documents into any panel. SHA-256 deduplication, inline preview, and OCR text extraction built in.

⚡

On-Device AI Summary

One-click local AI (Ollama) summarises any panel's content — private, on-device, zero cloud exposure for sensitive data.

🔍

Multi-Criteria Search

Tagify-powered search across all notes and items. ⌘K opens an instant modal — Spotlight for your entire knowledge base.

☁️

Google Drive Integration

OAuth-backed Drive sync: upload, browse, and link your identity — no tab-switching to find a document ever again.

⌨️

Keyboard-First UX POWER

Comprehensive shortcuts. ⌘K to search, ⌘W for new panel, ⌘I for workspaces. Built for zero-friction, high-velocity work.

🖥️

Web + Desktop Dual Target

Same codebase ships as a web app on AWS Amplify and as a native Electron desktop app for macOS, Windows, and Linux.

How It Works

Four steps to clarity.

01

Create a Workspace

Group knowledge into sections — projects, domains, clients. Pin the ones you need front and centre.

02

Add Topics & Items

Each item is a node in your knowledge graph. Drag to sub-topics, tag them, or bulk-import instantly.

03

Open Note Panels

Click any item to open a floating note panel. Write, attach files, paste images — everything in one place.

04

Search & Summarise

⌘K searches everything instantly. AI summarise generates a condensed panel in one click — privately.

Future Scope

The 5-year vision.

Prep-Arch isn't just a notes app — it's infrastructure for how professionals think and build. Here's where the platform goes.

AI Layer

Contextual AI That Knows Your Workspace

Unlike generic AI, Prep-Arch's AI engine will have full context of your topics, notes, and history. Ask "What did I decide about the API architecture last month?" and get an accurate answer — on-device for privacy, cloud for power users who opt in.

Collaboration

Real-Time Multiplayer Workspaces

Every topic and panel becomes a live collaboration surface. Presence indicators, inline comments, cursor sharing — like Figma, but for knowledge work. Team workspaces, org-wide search, and granular permissions for enterprise rollout.

Platform

Integration Ecosystem & Open API

A marketplace where developers build connectors: Jira-to-items, Slack-to-panels, GitHub issues as topics. An open API makes Prep-Arch the operating layer for any team's knowledge graph, with revenue-share creating sustainable moat.

Vertical SaaS

Legal, Medical & Education Plays

The workspace-panel-vault model maps naturally to case management (legal), patient records (healthcare), and study systems (education). Vertical packages with HIPAA / GDPR compliance and domain-specific AI unlock $B+ adjacent markets.

Agents

Autonomous Knowledge Agents

AI agents that monitor your workspace and proactively surface insights: "You have 12 unsorted items about Q3 planning — want me to organise them?" Knowledge hygiene, automated — without leaving the app.

Mobile

iOS & Android: Capture Anywhere

Native mobile apps for quick item creation from voice, camera, or clipboard — synced instantly to your desktop workspace. Offline-first with smart conflict resolution. Capture a meeting insight on your phone; it's in the right workspace before you sit down.
